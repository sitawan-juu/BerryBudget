# 🌸 SugarLogic: Python-Based Financial Tracker

**SugarLogic** is an intuitive, data-driven personal finance assistant built with Python. It is designed to bridge the gap between simple manual bookkeeping and meaningful data analysis, providing users with a clear visual overview of their spending habits.

---

## 🌟 Features

* **🌸 Seamless Logging:** Quickly record income and expenses with automated timestamps and custom notes.
* **📊 Smart Analytics:** Powered by **Pandas** for high-speed data aggregation and real-time balance calculations.
* **📈 Visual Insights:** Generates clean, pastel-themed bar charts via **Matplotlib** to compare inflows and outflows.
* **💾 Robust Persistence:** Automatically saves transactions to an Excel-friendly CSV file (`money_data.csv`).
* **🔙 Undo Function:** Easily remove the most recent entry to correct mistakes instantly.
